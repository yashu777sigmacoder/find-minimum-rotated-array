# Contributing to Find Minimum in Rotated Array

Contributions are welcome! Here's how you can help:

## Ways to Contribute

1. **Add More Test Cases** - Help improve test coverage
2. **Optimize Algorithms** - Suggest performance improvements
3. **Documentation** - Improve or add to the guides
4. **New Approaches** - Suggest alternative algorithms
5. **Bug Reports** - Report any issues you find

## Getting Started

1. Fork the repository
2. Create a feature branch: `git checkout -b feature/your-feature`
3. Make your changes
4. Test thoroughly: `python test_find_min_rotated.py`
5. Commit: `git commit -am 'Add your feature'`
6. Push to the branch: `git push origin feature/your-feature`
7. Submit a Pull Request

## Coding Standards

- Follow PEP 8 conventions
- Add docstrings to all functions
- Include time and space complexity in docstrings
- Add test cases for new functionality
- Keep comments clear and concise

## Questions?

Feel free to open an issue or reach out!
